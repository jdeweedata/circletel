## 1. Executive Summary

CircleTel Venue+ combines CloudWiFi managed WiFi-as-a-Service with Arlan MTN IoT connectivity for commercial venues. The product addresses the growing need for integrated guest WiFi and IoT infrastructure in retail, hospitality, and commercial spaces.

By bundling venue WiFi with IoT SIMs for POS systems, sensors, and backup connectivity, Venue+ creates a single-vendor solution with the highest margins in the CircleTel portfolio (40%+).

### 1.1 Strategic Metrics Summary

| Metric | Value | Notes |
|--------|-------|-------|
| Strategic Priority | ★★★ | High-margin vertical |
| Target Market | Commercial venues | Retail, hospitality, office |
| Base Margin | 40-44% | Before commissions |
| Technology | WiFi 6 + LTE IoT | Reyee + MTN IoT SIMs |
| Coverage | National | Venue-based deployment |
| Pricing Model | Tiered + Modular | 3 tiers + add-on modules |

## 2. Product Architecture

Venue+ combines CloudWiFi managed services with Arlan MTN IoT SIMs for connected devices.

### 2.1 Bundle Components

| Component | Provider | Role |
|-----------|----------|------|
| Managed WiFi | CircleTel (CloudWiFi) | Guest + operational WiFi |
| IoT Connectivity | Arlan (MTN IoT SIMs) | POS, sensors, backup |
| Cloud Management | Ruijie Cloud | Zero-license monitoring |

### 2.2 Tier Structure

| Tier | CloudWiFi | IoT SIMs | Target Venue | Bundle Price |
|------|-----------|----------|--------------|--------------|
