## 3. Service Portfolio

Echo SP SA (Pty) Ltd provides the following services to Circle Tel SA (Pty) Ltd:

### 3.1 Managed BNG (Broadband Network Gateway) Service

**Service Description:** Echo SP provides a fully managed Broadband Network Gateway service, including RADIUS proxy services, realm-based routing, integration with Interstellio AAA platform, subscriber session management, and accounting/reporting.

**Service Schedule Reference:** SS Q27988  
**Quote Date:** 06 August 2025  
**Initial Period:** 12 months

#### Managed BNG Service Inclusions

The Managed BNG Service includes the following components:

- RADIUS proxy services
- Realm-based routing (circletel.co.za)
- Integration with Interstellio AAA platform
- Subscriber session management
- Accounting and reporting
- 24/7 NOC support

### 3.2 Internet Access / IP Transit

**Service Description:** Blended IP transit via multiple Tier 1 carriers at Teraco data centres, providing carrier-grade internet connectivity with redundancy.

### 3.3 CGNAT (Carrier-Grade Network Address Translation)

**Service Description:** NAT processing service enabling multiple subscribers to share public IP addresses, reducing IPv4 address requirements whilst maintaining internet connectivity.

### 3.4 Static IP Address Services

**Service Description:** Public IPv4 address allocation services for business customers requiring dedicated IP addresses for hosting, remote access, or other services requiring static addressing.

### 3.5 Infrastructure Services at Teraco

**Service Description:** Echo SP maintains infrastructure at Teraco data centres (JB1 Johannesburg and CT1 Cape Town), providing colocation, cross-connect management, and carrier-neutral connectivity services.

**Infrastructure Locations:**

| Facility | Location | Service |
|----------|----------|---------|
| Teraco JB1 | Isando, Johannesburg | Primary colocation and interconnection |
| Teraco CT1 | Cape Town | Secondary site for redundancy |
