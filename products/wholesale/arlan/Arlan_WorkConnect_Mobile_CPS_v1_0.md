## 1. Executive Summary

WorkConnect + Mobile is CircleTel's primary cross-sell bundle for WorkConnect SOHO customers. It combines CircleTel's own fixed wireless internet service (delivered via Tarana FWB on the MTN Wholesale network) with one or more Arlan/MTN mobile device or connectivity deals, billed as a single account with a unified invoice.

**Strategic rationale**:

The bundle serves two simultaneous strategic objectives. First, it captures additional revenue per subscriber by layering zero-CAPEX Arlan commission and markup income on top of an already-profitable WorkConnect subscription. Second, each WorkConnect (Tarana) line added directly reduces CircleTel's Monthly Spend Commitment (MSC) exposure: Q3 2026 MSC is R29,940/mo, requiring 60 active Tarana subscribers at the R499 wholesale rate to break even. Every WorkConnect + Mobile bundle sold accelerates progress toward the Month 5 (August 2026) target of 100 Tarana customers, the point at which MSC coverage reaches 1.0x.

**Cross-sell rule**: Every new WorkConnect signup must be offered this bundle before order completion. The offer is mandatory; acceptance is optional.

**Revenue summary**:

| Component | Monthly Earning |
|-----------|----------------|
| WorkConnect Tarana margin (tier-dependent) | R153–R653/mo |
| Arlan commission + markup (add-on dependent) | R87–R100/mo |
| **Combined bundle earnings** | **R635–R1,035/mo** |

## 7. Network & Technical Specifications

### 7.1 WorkConnect — Tarana Fixed Wireless Broadband

| Parameter | Specification |
|-----------|--------------|
| Technology | Tarana G1 Fixed Wireless Access |
| Spectrum | Licensed 3.5GHz CBRS-class or 5.8GHz (site-dependent) |
| Line of Sight | Required (LoS preferred; NLoS up to 3km if path clear of major obstructions) |
| Max range | 8km with direct LoS |
| Contention ratio | 4:1 shared |
| Backhaul | MTN NNI via CrossConnect at Teraco JHB1/CT2 PoP |
| IP assignment | Dynamic (CGNAT) or Static (+R50/mo) |
| IPv6 | Not currently supported |
| Latency (typical) | 8–25ms to CircleTel PoP |
| Coverage tool | 4-layer fallback: MTN WMS → MTN Consumer → Provider APIs → Mock |

### 7.2 Arlan Add-On — MTN Mobile (4G/5G LTE)

| Parameter | Specification |
|-----------|--------------|
| Technology | MTN 4G LTE / 5G NR (SA/NSA) |
| Coverage | MTN national network (95%+ population coverage) |
| IP assignment | Dynamic (MTN-managed) |
| SIM type | Standard/Micro/Nano (device-dependent) |
| IMEI lock | Device-locked to MTN for 24/36-month contract term |
| Provisioning | MTN-handled; SIM delivery 2–5 business days post-FICA |
| Roaming | Domestic roaming included; international roaming not included in base plan |

### 7.3 Dual-Link Resilience (Config C — Shesha Backup)

Where the customer takes WorkConnect Pro/Ultra + Shesha Backup, the Tozed CPE can be configured as a failover link:

- Primary: Tarana FWB (low latency, high throughput)
- Failover: Shesha 4G/5G LTE (MTN mobile network)
- Failover trigger: Manual (via router settings) — automatic failover requires compatible router (not included in standard installation)
- CircleTel recommends Mikrotik hAP ax³ or similar dual-WAN router for automatic failover (quoted separately as add-on hardware)
