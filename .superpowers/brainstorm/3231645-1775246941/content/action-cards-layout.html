How many quick-action cards on the dashboard?

SuperSonic has 6. CircleTel has more features (Usage, KYC, Compliance, Upgrade/Downgrade). Three options for handling this:

A

6 Primary Cards (same as SuperSonic)

Pay Now · Invoices & Statements · Update Banking · My Profile · Log a Ticket · Get Help. Extra features (Usage, KYC, Upgrade) accessible only via the top-nav user menu or directly from the service card's Manage dropdown.

Pros

  • Cleanest dashboard
  • No decision fatigue

Cons

  • Usage & KYC buried
  • Less discoverable
B

8 Cards (2×4 grid) — Recommended

Add Usage and Upgrade to the 6 SuperSonic cards. Pay Now · Invoices · Update Banking · My Profile · Log a Ticket · Get Help · Check Usage · Upgrade Plan. KYC/Compliance surfaced via onboarding banner only when needed.

Pros

  • Key differentiators visible
  • Still clean & scannable
  • 4-column grid looks balanced

Cons

  • Slightly more visual weight
C

Grouped Sections (Primary + Secondary)

6 primary cards at top. Below the service card, a secondary "Account Tools" row with smaller cards for Usage, KYC, Compliance, and Upgrade — visually separated with a section label.

Pros

  • Nothing hidden
  • Clear visual hierarchy

Cons

  • Longer page
  • More complex to build
