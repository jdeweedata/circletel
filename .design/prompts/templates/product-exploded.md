## Variables to Customize

| Placeholder | Options |
|-------------|---------|
| `[product]` | router, mesh node, CPE, ONT, antenna |
| `[components]` | electronics, mechanics, antenna array |
| `[core element]` | PCB, main board, chassis |
| `[internal parts]` | chips, capacitors, antennas, heatsinks |
| `[accent color]` | orange (#F5831F), copper, gold |

## Tips

1. **Layer order matters** - largest/outermost at top, core at bottom
2. **Vertical alignment** is crucial for the exploded effect
3. **Orange accents** on key components draw attention
4. **No readable text** - avoid chip labels or markings
5. **Consistent spacing** between layers (equal gaps)
6. **Shadows** should be subtle and centered under components
