# Product Split Design (Realistic + Digital)

> Half photorealistic product, half digital ecosystem visualization

## Pattern Overview

Creates a conceptual split where one half shows the physical product in photorealistic detail, while the other half shows the digital ecosystem, UI elements, or technical schematics. Perfect for hardware-software integration messaging.

## Template

```
[Product] centered and fully visible floating against clean
[background] background, conceptually split by design with
vertical divide, left half photorealistic showing actual
materials textures and reflections, right half flat digital
elements including [digital elements], holographic UI
fragments floating around product, schematic overlays on
digital side, [brand color] accent glow at split line,
clean modern composition, commercial product photography
meets digital design, 4K, [aspect ratio]
```

## Variables

| Variable | Description | Examples |
|----------|-------------|----------|
| `[Product]` | Hardware item | "5G CPE router", "mesh WiFi node", "fiber ONT" |
| `[background]` | Backdrop color | "white", "dark charcoal (#1F2937)", "gradient" |
| `[digital elements]` | UI/tech visuals | "app interfaces, WiFi signals, data streams" |
| `[brand color]` | Accent | "orange (#F5831F)" |
| `[aspect ratio]` | Output | "16:9", "1:1", "4:3" |

## Key Elements

| Element | Purpose |
|---------|---------|
| `floating against clean` | Isolates product |
| `conceptually split` | Dual narrative |
| `photorealistic left` | Physical credibility |
| `flat digital right` | Software/ecosystem |
| `accent glow at split` | Visual connection |
| `holographic overlays` | Futuristic but grounded |

## Split Styles

| Style | Description | Best For |
|-------|-------------|----------|
| **Vertical 50/50** | Clean center divide | Feature pages |
| **Diagonal** | Dynamic energy | Hero banners |
| **60/40** | Emphasize one side | Product focus |
| **Fade blend** | Gradual transition | Subtle messaging |

## Post-Processing

1. Generate base split image
2. Ensure brand color (#F5831F) is prominent at split
3. Add CircleTel logo (can place on digital side)
4. Check text overlay space for copy
5. Export in multiple aspect ratios
