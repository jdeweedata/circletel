# Photo Mosaic Grid Composite

> Grid-based portrait filled with milestone/career moments

## Pattern Overview

Creates tribute-style posters where a central silhouette contains a dense grid of smaller photos representing journey, milestones, or community. Perfect for company anniversaries and customer success stories.

## Template

```
Portrait silhouette filled with dense photo mosaic grid,
central figure outline of [subject description] as container,
interior filled with [number] small square photographs
showing [theme], varied texture treatments including halftone
dots and subtle film grain per cell, monochrome black and
white base with selective [accent color] color overlays on
key images, typography "[TEXT]" in Inter Semibold tight
kerning at [position], MAX 20% canvas width for text,
tribute poster style, editorial composite, clean edges,
4K, [aspect ratio]
```

## Variables

| Variable | Description | Examples |
|----------|-------------|----------|
| `[subject description]` | Silhouette figure | "standing professional", "walking person", "seated figure" |
| `[number]` | Grid density | "50", "100", "200" |
| `[theme]` | Photo content | "career milestones and achievements", "customer interactions" |
| `[accent color]` | Brand highlight | "orange (#F5831F)", "blue" |
| `[TEXT]` | Optional headline | "10 YEARS", "OUR STORY" |
| `[position]` | Text placement | "bottom center", "top left" |
| `[aspect ratio]` | Output size | "9:16", "16:9", "1:1" |

## Key Elements

| Element | Purpose |
|---------|---------|
| `silhouette as container` | Unifies diverse images |
| `halftone dots, film grain` | Adds texture variety |
| `B&W with selective color` | Focus on key moments |
| `Inter Semibold tight kerning` | Modern professional type |
| `MAX 20% canvas width` | Prevents oversized text |

## Grid Density Guide

| Density | Photos | Best For |
|---------|--------|----------|
| Sparse | 30-50 | Large individual moments visible |
| Medium | 80-100 | Balance of detail and pattern |
| Dense | 150-200 | Textural effect, less individual recognition |

## Texture Treatments

- **Halftone dots**: Newspaper/printing aesthetic
- **Film grain**: Vintage photo feel
- **Fabric/embroidery**: Premium tactile quality
- **Clean squares**: Modern minimal

## Post-Processing

1. Generate base mosaic composite
2. Add CircleTel logo in corner (white or orange)
3. Ensure text is legible against grid
4. Export in multiple sizes
