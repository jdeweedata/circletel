# Character Product Photography

> Anthropomorphic product characters for engaging social content

## Pattern Overview

Creates cute, character-based versions of products with faces and accessories. Perfect for friendly social media engagement, younger audiences, and approachable brand messaging.

## Template

```
8K ultra high definition character product photography,
[product] reimagined as adorable [character type] with
[facial features], wearing [accessories/props] related
to [activity], posed in [action/position], warm cinematic
lighting with golden bokeh background, soft shadows,
friendly approachable aesthetic, Pixar-quality character
design, photorealistic textures, cute expressive eyes,
no AI-generated text, 4K, [aspect ratio]
```

## Variables

| Variable | Description | Examples |
|----------|-------------|----------|
| `[product]` | Base product | "WiFi router", "ethernet cable", "mesh node" |
| `[character type]` | Creature style | "friendly robot", "cute animal", "cartoon figure" |
| `[facial features]` | Face elements | "small round eyes, gentle smile" |
| `[accessories/props]` | Props | "tiny hard hat", "miniature laptop", "small headphones" |
| `[activity]` | Related use | "internet browsing", "streaming video", "gaming" |
| `[action/position]` | Character pose | "waving hello", "giving thumbs up", "running excitedly" |
| `[aspect ratio]` | Output | "1:1", "4:5", "9:16" |

## Character Design Tips

### Facial Features
- **Eyes**: Round, slightly oversized, reflective
- **Smile**: Gentle curve, not too wide
- **Expressions**: Happy, curious, helpful
- **LED integration**: Use existing lights as eyes

### Props/Accessories
- **Size**: Miniature versions of related items
- **Relevance**: Connect to usage/benefit
- **Brand color**: Include orange (#F5831F) elements
- **Texture**: Match product material quality

### Poses
- **Waving**: Welcoming, social engagement
- **Thumbs up**: Approval, satisfaction
- **Helping**: Offering assistance
- **Celebrating**: Achievement, success

## Post-Processing

1. Generate character image
2. Add CircleTel logo badge or watermark
3. Ensure brand orange (#F5831F) is visible
4. Create series with consistent style
5. Size for social platforms
