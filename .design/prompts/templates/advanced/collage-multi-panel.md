# Multi-Panel Editorial Collage

> Asymmetric photo grid with floating UI elements

## Pattern Overview

Creates editorial-style compositions with multiple photo panels unified by consistent aesthetic, floating UI overlays, and subtle design treatments. Perfect for customer journeys, product stories, and multi-angle features.

## Template

```
Editorial multi-panel collage with [number] asymmetric
photographic frames, each panel showing [subject] from
different [angle/moment/stage], unified by consistent
[color palette] warm amber neutral palette, floating
translucent UI elements including [UI type] overlays,
subtle grid lines and cursor-like frame accents, each
panel with slightly different crop and composition,
realistic lens distortion on edges, film photography
aesthetic with fine grain, magazine editorial layout,
clean white gaps between panels, 4K, [aspect ratio]
```

## Variables

| Variable | Description | Examples |
|----------|-------------|----------|
| `[number]` | Panel count | "3", "4", "6" |
| `[subject]` | Content theme | "product unboxing stages", "customer journey" |
| `[angle/moment/stage]` | Variation | "angles", "time progression", "usage contexts" |
| `[color palette]` | Tonal range | "warm amber neutral", "cool blue professional" |
| `[UI type]` | Overlay style | "iOS-style music player", "speed dashboard", "app interface" |
| `[aspect ratio]` | Output | "16:9", "1:1", "9:16" |

## Post-Processing

1. Generate multi-panel composite
2. Ensure consistent color grading across panels
3. Add CircleTel logo watermark (corner or panel)
4. Verify UI overlays are legible but subtle
5. Add brand orange (#F5831F) accent if needed
6. Export in multiple aspect ratios

## Aspect Ratios

| Use Case | Ratio | Panel Layout |
|----------|-------|--------------|
| Website Banner | 16:9 | 3-4 horizontal |
| Social Post | 1:1 | 4 grid |
| Social Story | 9:16 | 3 vertical |
| Hero Section | 21:9 | 4-6 panoramic |
