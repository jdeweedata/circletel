# Exploded View Animation

> Video/motion prompt for dynamic product breakdown

## Pattern Overview

Creates motion content showing products exploding apart to reveal components, features, or ingredients. Perfect for video content, animated social posts, and product feature breakdowns.

## Template (Video/Motion)

```
[Duration] smooth animation of [product] exploding apart
in slow motion, starting from assembled form then
separating into [number] distinct [component type],
components floating and rotating to reveal [detail focus],
[lighting style] with [brand color] accent lighting on
key features, particle effects and [motion element]
during separation, camera [camera movement], clean
[background] background, premium product reveal aesthetic,
cinematic motion graphics quality, 4K [frame rate],
[aspect ratio]
```

## Variables

| Variable | Description | Examples |
|----------|-------------|----------|
| `[Duration]` | Video length | "3 second", "5 second", "10 second" |
| `[product]` | Item to explode | "5G router", "mesh node", "ONT device" |
| `[number]` | Component count | "5", "8", "12" |
| `[component type]` | Parts description | "technical components", "layers", "modules" |
| `[detail focus]` | What to highlight | "internal circuitry", "antenna arrays", "cooling system" |
| `[lighting style]` | Light quality | "dramatic studio", "soft diffused", "rim lighting" |
| `[brand color]` | Accent | "orange (#F5831F)" |
| `[motion element]` | VFX | "light trails", "energy particles", "data streams" |
| `[camera movement]` | Camera | "orbit around", "push in slowly", "static centered" |
| `[background]` | Backdrop | "white seamless", "dark gradient", "abstract tech" |
| `[frame rate]` | FPS | "24fps", "30fps", "60fps" |
| `[aspect ratio]` | Output | "16:9", "9:16", "1:1" |
