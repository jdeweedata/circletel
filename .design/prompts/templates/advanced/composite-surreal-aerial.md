# Surreal Aerial Composite

> Giant products in real-world landmark locations

## Pattern Overview

Creates attention-grabbing composites placing oversized products in recognizable real-world locations. Perfect for viral social content, brand awareness campaigns, and memorable advertising.

## Template

```
Aerial photograph composite with giant [product] placed
at [landmark/location], [interaction element] adjusting
or interacting with product, real-world backdrop showing
recognizable [location features], golden [time of day]
light with long dramatic shadows, product reflecting
surrounding environment, [weather/atmosphere], surreal
scale juxtaposition, hyper-realistic product materials
against real photography, seamless composite photography,
national geographic documentary quality, 4K, [aspect ratio]
```

## Variables

| Variable | Description | Examples |
|----------|-------------|----------|
| `[product]` | Oversized item | "5G router", "mesh WiFi node", "fiber cable" |
| `[landmark/location]` | SA location | "Table Mountain", "Johannesburg skyline", "Cape Town V&A" |
| `[interaction element]` | Human touch | "giant hand from clouds", "construction crane", "helicopter" |
| `[location features]` | Identifying elements | "mountain silhouette", "city towers", "ocean harbor" |
| `[time of day]` | Lighting mood | "sunset", "golden hour", "blue hour dawn" |
| `[weather/atmosphere]` | Environmental | "clear sky", "dramatic clouds", "morning mist" |
| `[aspect ratio]` | Output | "16:9", "4:5", "1:1" |

## Post-Processing

1. Generate base composite image
2. Verify landmark is recognizable
3. Add CircleTel logo (building billboard, sky banner)
4. Enhance orange (#F5831F) on product
5. Add caption space for social copy
6. Create versions for different platforms
