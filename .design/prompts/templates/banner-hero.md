# Website Hero Banner Template

> Large hero sections for website landing pages

## Pattern

```

[brand color integration], space for [text overlay position],
[mood and tone], cinematic wide composition, [background
treatment], web marketing aesthetic, [resolution], [aspect_ratio]
```

## Variables to Customize

| Placeholder | Options |
|-------------|---------|
| `[visual subject]` | abstract speed, family, product, office |
| `[composition style]` | left-weighted, centered, right-weighted |
| `[text overlay position]` | left third, bottom, center-left |
| `[mood and tone]` | premium, warm, professional, exciting |
| `[background treatment]` | gradient, lifestyle blur, abstract |

## Text Space Guidelines

| Hero Type | Text Position | Safe Area |
|-----------|---------------|-----------|
| Abstract | Left 40% | Avoid bright areas |
| Lifestyle | Left 40% | Use overlay if needed |
| Product | Left 35% | Product on right |
| Map | Left 30% | Map can be background |

## Tips

1. **21:9 is standard** - ultra-wide hero format
2. **Left-side text** - most heroes have copy on left
3. **Dark gradients** - ensure text readability
4. **No focal points** - where text will overlay
5. **Mobile crop** - design for center-crop on mobile
6. **Orange accents** - brand color should appear
7. **South African context** - local when appropriate
