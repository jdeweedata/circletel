## Variables to Customize

| Placeholder | Options |
|-------------|---------|
| `[Social media format]` | Instagram, Facebook, Twitter, LinkedIn |
| `[subject/offer]` | speed deal, bundle, new service, promotion |
| `[brand color scheme]` | orange primary, charcoal background |
| `[visual style]` | abstract, lifestyle, product, iconographic |
| `[text placement]` | center, right side, bottom |
| `[mood]` | urgent, exciting, professional, friendly |

## Aspect Ratio Reference

| Platform | Format | Ratio |
|----------|--------|-------|
| Instagram Post | Square | 1:1 |
| Instagram Story | Vertical | 9:16 |
| Facebook Post | Landscape | 16:9 |
| Facebook Ad | Landscape | 1.91:1 |
| Twitter Post | Landscape | 16:9 |
| Twitter Card | Wide | 2:1 |
| LinkedIn Post | Landscape | 1.91:1 |

## Tips

1. **Leave text space** - these need headlines and CTAs added
2. **Brand colors always** - orange #F5831F on charcoal #1F2937
3. **Platform-specific** - match the exact aspect ratio
4. **Visual hierarchy** - direct eye to where text will go
5. **Not too complex** - will compete with overlaid text
6. **Mobile-first** - most views on small screens
