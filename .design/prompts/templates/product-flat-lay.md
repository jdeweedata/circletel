## Variables to Customize

| Placeholder | Options |
|-------------|---------|
| `[product]` | router, mesh node, ONT, antenna |
| `[accessories]` | cables, adapters, documentation, tools |
| `[material finish]` | matte, glossy, textured, brushed metal |
| `[detail elements]` | LED glow, ventilation, ports |
| `[accent color]` | orange (#F5831F), white, black |
| `[background]` | pure white, light gray (#E6E9EF), dark charcoal |
| `[resolution]` | 2K, 4K, 8K |
| `[aspect_ratio]` | 1:1, 4:3, 16:9 |

## Tips

1. **Arrange accessories** to create visual flow toward the main product
2. **Use brand orange** on at least one accessory (cables work well)
3. **Keep spacing** consistent between items
4. **Shadows** should all fall in the same direction
5. **No text** should be legible - avoid instruction cards with readable text
