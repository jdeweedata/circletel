# Coverage Visualization Template

> Signal strength, network coverage, connectivity range concepts

## Pattern

```
Abstract visualization of [coverage concept], [signal representation]
emanating from [source], [coverage metaphor] spreading across
[environment], [brand colors] gradient, sense of [emotion],
[background treatment], clean composition, [resolution], [aspect_ratio]
```

```yaml
  - Outer rings: faint orange, fading
  - Smooth gradient transitions

  Mood:
  - Comprehensive coverage
  - Reliable signal everywhere
  - Home protection
  - Connected family

  Style: Abstract tech visualization, radial design,
  clean modern, orange on dark, 4K, 16:9.
```

## Variables to Customize

| Placeholder | Options |
|-------------|---------|
| `[coverage concept]` | WiFi range, network reach, signal strength |
| `[signal representation]` | waves, circles, gradients, bars |
| `[source]` | router, tower, mesh node |
| `[coverage metaphor]` | waves, zones, clouds, fields |
| `[environment]` | home, office, neighborhood, city |
| `[emotion]` | protection, connection, reliability |

## Tips

1. **Concentric circles** are universally understood as signal/coverage
2. **Gradient from center** shows signal degradation intuitively
3. **Orange (#F5831F)** should be brightest at signal source
4. **Device icons** help show practical coverage benefits
5. **South African context** adds local relevance when appropriate
6. **Leave text space** - coverage images often need explanatory copy
