# Speed Visualization Template

> Abstract representations of internet speed, performance, data flow

## Pattern

```
Abstract visualization of [concept], streams of [brand color] light
particles flowing through [contrasting color] space, [technology
metaphor], data packets visualized as [element], sense of [emotion],
cinematic depth of field, futuristic but approachable, clean
composition for text overlay on [position], [resolution], [aspect_ratio]
```

## Variables to Customize

| Placeholder | Options |
|-------------|---------|
| `[concept]` | speed, data flow, connectivity, bandwidth |
| `[brand color]` | orange (#F5831F), warm orange gradient |
| `[contrasting color]` | dark charcoal (#1F2937), deep blue, black |
| `[technology metaphor]` | fiber optics, data streams, light trails |
| `[element]` | glowing dots, light nodes, energy particles |
| `[emotion]` | power, reliability, speed, freedom |
| `[position]` | left, right, bottom, corners |

## Tips

1. **Orange on dark** is the primary visual motif for CircleTel speed
2. **Leave space** for text overlay - plan for marketing copy
3. **Motion blur** suggests speed better than static images
4. **Depth of field** creates dimension in abstract visuals
5. **Avoid complexity** - one clear concept per image
6. **Horizontal flow** (left to right) feels natural and positive
