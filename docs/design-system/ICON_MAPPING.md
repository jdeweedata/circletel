## Usage

Import Phosphor icons from `react-icons/pi`:

```typescript
import { PiCheckBold, PiXBold, PiUserBold } from 'react-icons/pi';
```

## Weight Variants

Phosphor offers multiple weights. We use **Bold** as the default:
- `PiUser` - Regular
- `PiUserBold` - Bold (our default)
- `PiUserFill` - Filled
- `PiUserDuotone` - Two-tone
