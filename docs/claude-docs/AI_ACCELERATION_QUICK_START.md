## ğŸš¨ Troubleshooting

### Skill Not Working
1. Check skill exists: `ls .claude/skills/`
2. Verify SKILL.md has frontmatter
3. Restart Claude Code

### Sub-Agent Not Responding
1. Check agent config: `.claude/agents/<agent-name>.yml`
2. Verify tools are available
3. Check logs for errors

### Plugin Not Loading
1. Verify installation: `npm list <plugin-name>`
2. Check configuration: `.claude/config.json`
3. Restart Claude Code
